# System Flow

## Creator Flow

1. Register as CREATOR
2. Login and create campaign
3. Upload campaign image/media
4. Add reward tiers
5. Move status to ACTIVE
6. Monitor pledges from dashboard
7. Request payout after success

## Backer Flow

1. Register as BACKER
2. Browse campaigns
3. Open campaign detail
4. Create pledge
5. Complete sandbox checkout
6. Track pledges and notifications

## Admin Flow

1. Login as admin
2. Open Admin Panel
3. Review user reports
4. Move report status (OPEN -> UNDER_REVIEW -> RESOLVED/REJECTED)
5. Inspect audit logs
